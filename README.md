A collection of simple source code examples in Scala 3, featuring key aspects of the language.

## Build with SBT

 1. Install 'sbt' from here: http://www.scala-sbt.org/download.html
 2. Run a sbt shell with 
    
        cd [project dir]
        sbt
 3. Then, compile with 
    
        compile
 4. Then, run with
    
        run
 5. Then, execute tests with

        test
